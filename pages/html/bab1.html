🌐 Belajar HTML

Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa yang digunakan untuk membuat struktur sebuah website.

Contoh HTML

<!DOCTYPE html>
<html>
<head>
<title>Hello</title>
</head>

<body>
<h1>Halo Dunia</h1>
</body>

</html>

Latihan

Buatlah sebuah heading menggunakan tag h1.
